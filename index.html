PDF Organizer

다운로드된 PDF를 기존 폴더 또는 !zz_Singles 기준으로 정리합니다.

1. 새 다운로드 폴더

2. 기존 패턴 폴더

!로 시작하는 관리용 폴더는 디자이너 목록에서 제외합니다. 단, !zz_Singles 안의 PDF는 세 번째 패턴 판별에 사용합니다.

3. 새 PDF 정리 미리보기

기존 폴더를 선택하거나 새 저장 기준 이름을 입력하세요. 파일명 속 디자이너 표기를 입력하면 패턴 파일명을 자동 정리합니다.

자동 정리 대상: Designer_Pattern, Designer_-_Pattern, Pattern_Designer, Pattern by Designer
새 디자이너의 기존 Singles 파일과 이번 다운로드 파일을 합쳐 3개 이상이면 새 폴더 생성 대상으로 표시합니다.

PDF Organizer

다운로드된 PDF를 기존 폴더 또는 !zz_Singles 기준으로 정리합니다.

1. 새 다운로드 폴더

2. 기존 패턴 폴더

!로 시작하는 관리용 폴더는 디자이너 목록에서 제외합니다. 단, !zz_Singles 안의 PDF는 세 번째 패턴 판별과 중복 검사에 사용합니다.

3. 새 PDF 정리 미리보기

기존 폴더를 선택하거나 새 저장 기준 이름을 입력하세요. 입력이 끝나면 중복 검사를 실행한 뒤 정리 실행 버튼을 누릅니다.

자동 파일명 정리: Designer_Pattern, Designer_-_Pattern, Pattern_Designer, Pattern by Designer

같은 최종 파일명이 있지만 내용이 다르면 해당 파일만 보류합니다.

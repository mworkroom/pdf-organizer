PDF Organizer

다운로드된 PDF를 기존 디자이너 폴더 또는 !zz_Singles 기준으로 정리합니다.

1. 새 다운로드 폴더

2. 기존 패턴 폴더

!로 시작하는 관리용 폴더는 디자이너 목록에서 제외합니다. 단, !zz_Singles 안의 PDF는 세 번째 패턴 판별과 중복 검사에 사용합니다.

기억된 디자이너 연결

기억된 연결은 이 브라우저에 저장됩니다. 잘못 저장한 연결은 삭제할 수 있습니다.

3. 새 PDF 정리 미리보기

기존 디자이너를 선택하거나 새 저장 기준 이름을 입력하세요. 지금 처리하지 않을 파일은 ‘이번 작업에서 제외’를 체크하면 다운로드 폴더에 그대로 남습니다.

자동 파일명 정리: Designer_Pattern, Designer_-_Pattern, Pattern_Designer, Pattern by Designer
검색과 파일 비교에서는 악센트 차이를 무시합니다: Descoteaux = Descôteaux.
완전히 같은 PDF가 이미 저장되어 있으면 파일명이 달라도 다운로드본만 삭제합니다.
같은 최종 파일명이 있지만 내용이 다르면 해당 파일만 보류합니다.
이번 작업에서 제외한 파일은 중복 검사, 이동, 삭제, 새 폴더 생성 계산에서 모두 제외됩니다.

4. 작업 로그

실제 정리 실행 기록만 저장됩니다. 최근 50회까지 이 브라우저에 남습니다.
